# Family History Research

Guide for family history and genealogy research projects.

## Getting Started

### 1. Gather What You Know

Document:
- Names (including maiden names)
- Birth/death dates and places
- Marriage dates and places
- Occupations
- Migration history
- Family stories and traditions

### 2. Organize Information

```markdown
# [Family Name] Family Tree

## Generation 1 (Oldest Known)
- **[Name]** (b. YYYY, d. YYYY)
  - Born: [Location]
  - Married: [Spouse], [Date]
  - Occupation: [Job]
  - Children: [list]

## Generation 2
...
```

### 3. Research Sources

**Free Resources:**
- FamilySearch.org
- FindAGrave.com
- USGenWeb.org
- Newspapers.com (limited free)
- Local library genealogy sections

**Paid Resources:**
- Ancestry.com
- MyHeritage.com
- Newspapers.com (full)
- Fold3.com (military)

### 4. Document Types

- Birth certificates
- Death certificates
- Marriage records
- Census records
- Immigration records
- Military records
- Church records
- Newspaper articles
- Wills and probate

## Research Tips

1. **Work backwards** - Start with what you know
2. **Verify sources** - Don't trust a single source
3. **Document everything** - Note sources for all facts
4. **Talk to relatives** - Living memories are valuable
5. **Consider name variations** - Spellings changed

## Citation Format

```
[Person Name], [Document Type], [Date], [Location]; 
accessed [Date] at [Source/URL].
```

## Common Challenges

| Challenge | Strategy |
|-----------|----------|
| Name changes | Check immigration records, census variants |
| Missing records | Try neighboring areas, different record types |
| Brick walls | DNA testing, hire researcher |
| Language barriers | Translation services, native speaker help |
