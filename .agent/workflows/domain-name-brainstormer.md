# Domain Name Brainstormer

Generate creative domain name ideas and check availability.

## Process

### 1. Understand the Project
- What is the product/service?
- Target audience?
- Key keywords?
- Tone (professional, playful, tech)?

### 2. Generate Ideas

**Strategies:**
- Combine keywords (DropBox, YouTube)
- Add prefixes (GetX, TryX, UseX)
- Add suffixes (Xly, Xify, Xio)
- Use abbreviations
- Made-up words (Google, Spotify)
- Descriptive phrases (Buffer, Slack)

### 3. Naming Patterns

| Pattern | Example |
|---------|---------|
| Verb + Noun | Dropbox, Pushbullet |
| Noun + ly | Grammarly, Bitly |
| Noun + ify | Spotify, Shopify |
| Get/Try + X | GetResponse, TryNow |
| X + io | Figma.io, Notion.io |
| Compound | YouTube, Facebook |
| Made-up | Google, Reddit |

### 4. Check Availability

**TLDs to Check:**
- .com (most valuable)
- .io (tech startups)
- .dev (developers)
- .ai (AI products)
- .co (alternative to .com)
- .app (applications)

**Tools:**
- namecheap.com
- godaddy.com
- instantdomainsearch.com

### 5. Evaluate Options

**Criteria:**
- Easy to spell
- Easy to pronounce
- Memorable
- Not too long (under 12 chars ideal)
- Available across key TLDs
- Available on social media

## Output Format

```markdown
# Domain Ideas for [Project]

## Top Recommendations
1. **example.com** - Available ✅
2. **getexample.io** - Available ✅

## Alternative Options
- example.dev
- tryexample.com

## Social Media Availability
- Twitter: @example ❌ (taken)
- Instagram: @example ❌ (taken)
```
