## Usage

### HTML/CSS
```html
<style>
:root {
  /* Apply theme variables */
}
</style>
```

### Presentations
Apply colors to:
- Backgrounds
- Headings
- Accent elements
- Charts/graphs

### Documents
- Heading colors
- Link colors
- Highlight colors
- Border colors
