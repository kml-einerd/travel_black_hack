# Content Research Writer

Collaborative writing assistant for high-quality content with research.

## Workflow

### 1. Understand the Project
- What's the topic and main argument?
- Who's the target audience?
- What's the desired length/format?
- What's your goal? (educate, persuade, entertain)
- Any existing research or sources?

### 2. Create Outline

```markdown
# Article: [Title]

## Hook
- [Opening line/story/statistic]

## Introduction
- Context and background
- Problem statement

## Main Sections

### Section 1: [Title]
- Key points
- [Research needed: topic]

### Section 2: [Title]
- Key points
- [Citation needed: claim]

## Conclusion
- Summary
- Call to action
```

### 3. Conduct Research

**For each topic:**
- Search for relevant information
- Find credible sources
- Extract key facts, quotes, data
- Add citations

```markdown
## Research: [Topic]

Key Findings:
1. **Stat**: [Data point] [1]
2. **Quote**: "[Quote]" - [Expert] [2]

Citations:
[1] Author. (Year). "Title"
[2] Source. Publication.
```

### 4. Improve Hooks

**Hook Options:**
1. **Bold statement** - Provocative claim
2. **Story** - Personal anecdote
3. **Data** - Surprising statistic
4. **Question** - Thought-provoking

### 5. Section Feedback

For each section:
- What works well
- Clarity improvements
- Flow suggestions
- Evidence needs
- Specific line edits

### 6. Final Review

```markdown
# Full Draft Review

## Strengths
- [Key strengths]

## Structure & Flow
- [Assessment]

## Pre-Publish Checklist
- [ ] All claims sourced
- [ ] Citations formatted
- [ ] Transitions smooth
- [ ] Proofread complete
```

## Principles
- Preserve writer's voice
- Suggest, don't replace
- Research thoroughly
- Be specific with feedback
