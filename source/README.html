Entity Descriptions

Credit Cards
cards.json

Complete database of travel credit cards with issuers, earning rates, welcome bonuses, perks, and transfer partners.

Key Fields
  • id string
  • issuer_id FK‚Üíissuers
  • welcome_bonus object
  • annual_fee number
  • earning_rates[] array
  • perks[] array
  • transfer_partners[] array
Loyalty Programs
programs.json

Loyalty programs with real-time valuations, trends, health status, and historical data for tracking point values.

Key Fields
  • id string
  • current_value float (¬¢)
  • trend up|down|stable
  • health_status enum
  • transfer_partners[] array

Transfer Bonuses
transfer-bonuses.json

Active transfer bonus promotions between programs with expiration dates, effective ratios, and sweet spot recommendations.

Key Fields
  • from_program_id FK
  • to_program_id FK
  • bonus_percent integer
  • expires_at datetime
  • value_rating enum
Hacks & Strategies
hacks.json

Complete collection of points/miles strategies with difficulty ratings, step-by-step instructions, and community ratings.

Key Fields
  • category_id FK‚Üícategories
  • difficulty easy|medium|hard
  • steps[] array
  • requirements[] array
  • warnings[] array
  • legality enum
Quick Guides
guides.json

Educational guides covering award booking, credit card strategy, hotel status, and advanced redemption techniques.

Key Fields
  • category_id FK
  • difficulty beginner|intermediate|advanced
  • read_time integer (min)
  • sections[] array
  • action_items[] array
Notifications
notifications.json

Live notifications and alerts for transfer bonuses, card offers, award availability, and mistake fares.

Key Fields
  • type transfer-bonus|card-offer|...
  • priority urgent|high|medium|low
  • title string
  • expires_at datetime|null
  • action_url string

Developer Usage Guide

Fetching Data

All JSON files are available at ../../source/*.json from pages. Use fetch() to load data dynamically:

fetch('../../source/cards.json')

Joining Data

Use foreign keys to join entities. Example: cards ‚Üí programs via transfer_partners[] array, which contains program IDs.

Image Paths

All images are relative to page location. From pages: ../../assets/images/. Subfolders: cards/, travel/, destinations/, hotels/.

Real-time Updates

For live data, update JSON files and refresh. Each file has a meta.last_updated timestamp for cache busting.
